Installation and Uninstallation

Minimal requirements: Capella. Install Capella as described here (online documentation).

Read the Configuration and Extensibility (online documentation) section.

There are two options to install the Add-On:

  • Dropins: Unzip the file and copy it in the dropins folder of your Capella installation and restart Capella. To uninstall, just remove it from the dropins folder.
  • Update site: Help -> Install new software. Then' Add 'a new repository location and click Archive to select the update site zip file. Select a name and follow the installation wizard. Uncheck Group by category to see the feature to install. To uninstall, click Help -> About Capella and then Installation Details. Find and select the Add-On in the Installed Software tab and click Uninstall.

To check that the Add-On is correctly installed, click on Help -> About Capella -> Installation Details

Check that the Basic Requirement is installed.

Model Migration

As of Capella 7.0.0, Requirement has been extracted from Capella core, to a dedicated addon.

If you have a model from 6.1.0 or earlier, It is required to migrate the model with Basic Requirement installed. Old Requirement data will be migrated to Basic Requirement data, and the viewpoint will be activated.

If you want to remove any requirement data from your model, you need to install Basic Requirement and migrate your model beforehand. Then, it is necessary to uninstall Basic Requirement, and use "Model Detachment" on your aird file This will clean all data associated with Basic Requirement.

Unreferencing Basic Requirement from the Viewpoint Manager will not be enough and may lead to invalid models.
