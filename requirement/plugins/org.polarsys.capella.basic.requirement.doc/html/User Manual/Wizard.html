Requirement Manager

Basic Requirement offers lightweight requirement management capabilities. Allocation is performed through matrices or using the dedicated wizard. On any element, using the contextual menu, select the Requirements Manager as shown below to open the requirement allocation dialog.

A double pane dialog appears, proposing all requirements.

In this panel, the user can allocate requirements on selected element of the model. The result of the allocation can be seen in a dedicated matrix or in the Semantic Browser.

Note that this requirement allocation capability can be used for requirements in successive engineering phases, allowing refinement of requirements.
